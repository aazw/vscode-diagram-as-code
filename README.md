# Dev Container for Diagram as Code

## Installed

* Mermaid
  
  * Mermeid CLI
    ```bash
    mmdc -i samples/mermaid/001.mmd
    ```
  * Mermaid Live Editor

    http://localhost:48081

  * VSCode Preview    

* PlantUML

  * CLI
    ```bash
    plantuml samples/plantuml/001.puml
    ```
  * PlantUML Server
    
    http://localhost:48080
    
  * VSCode Preview
